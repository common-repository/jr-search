== Description ==

This advanced search plugin allows users to search your site - As well as many, many other popular search engines!

The plugin has support for your blog, Google, Yahoo, Bing, Ask.com, Cuil, Technorati, YouTube, Answers.com and Wikipedia!

Simply enable the widget and everything is setup! You have the option to change the title of the header to your desired text, to make it customized for you.

== Installation ==

1. Upload `jr-search.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Edit the JR Search Settings in your Administration Panel
4. Enable the widget in the Appearance --> Widgets menu.
